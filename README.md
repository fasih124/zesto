## 🚀 Overview

Zesto is a **production-grade mobile app landing page** built for a modern food delivery platform.  
It’s designed to **convert visitors into app downloads**, communicate trust instantly, and showcase the product experience in a visually engaging way.

This is not a generic SaaS template — it’s a **consumer-focused, high-conversion landing page** built with a strong emphasis on:

- Emotional design (hunger-driven visuals)
- Clear product storytelling
- Smooth, high-performance animations
- Mobile-first UX

---

## 🧠 The Goal

Most food delivery apps struggle with:

- Generic, boring landing pages  
- Poor storytelling of the actual product experience  
- Low conversion from visitors → app downloads  
- No clear differentiation from competitors  

The goal of this project was to design a landing page that:

- Instantly communicates value ("Hot food. Zero hassle.")
- Visually demonstrates the product (not just describe it)
- Feels premium, modern, and trustworthy
- Drives users to download the app

---

## ✨ The Solution

I built a **fully responsive, animated landing page** that focuses on:

- Showing the product through **interactive app mockups**
- Using **conversion-focused copywriting**
- Creating a **smooth, app-like browsing experience**
- Structuring content for **maximum engagement and clarity**

Instead of static screenshots, the UI is recreated using real components — making the experience feel alive and real.

---

## 🔑 Key Features

- **📱 Interactive Phone Mockups**  
  Custom-built reusable phone component that renders real UI screens inside — making the product feel tangible and real.

- **⚡ High-Performance Animations**  
  Framer Motion animations with staggered transitions to guide attention and improve engagement.

- **🍽️ Product-Focused UI Screens**  
  Realistic app screens (Home, Menu, Tracking, Rewards) built with Tailwind — no fake images.

- **🎯 Conversion-Optimized Sections**  
  Every section is designed to move the user toward one goal: downloading the app.

- **📊 Restaurant Partner Section**  
  Business-focused section showing how restaurants benefit — increasing perceived product value.

- **📱 Fully Responsive Design**  
  Mobile-first layout optimized for all devices.

## 🧠 Development Approach

This project was built using a **structured AI-assisted workflow**:

- Defined full UI/UX system before coding
- Broke development into modular components
- Built reusable UI patterns (PhoneMockup, App Screens)
- Focused on performance + scalability from the start

AI was used as a **productivity tool**, not a shortcut — helping maintain consistency and speed without sacrificing quality.
